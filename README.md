    <article class="markdown-body entry-content" itemprop="text"><h1><a href="#make-admin-panel-in-laravel-56-part-1-to-part-6" aria-hidden="true" class="anchor" id="user-content-make-admin-panel-in-laravel-56-part-1-to-part-6"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>make-admin-panel-in-laravel-5.6 (Part 1 to Part 6)</h1>
<p>Part 1: Make Admin Panel in Laravel 5.6</p>
<p>Youtube Tutorial :- <a href="https://www.youtube.com/watch?v=-t7oMHzIpBQ&amp;t=266s" rel="nofollow">https://www.youtube.com/watch?v=-t7oMHzIpBQ&amp;t=266s</a></p>
<p>Instructions :-</p>
<p>First we install Laravel 5.6</p>
<p>Simply follow our Youtube tutorial <a href="https://www.youtube.com/watch?v=-t7oMHzIpBQ" rel="nofollow">https://www.youtube.com/watch?v=-t7oMHzIpBQ</a> to install Laravel 5.6</p>
<p>We assume you have installed Laravel 5.6 or you can follow our another tutorial given above to install it.</p>
<p>Now we will search for some free admin panel template.</p>
<p>We are going to download Matrix Admin template here.</p>
<p>See in video, we have simply downloaded the HTML files of Matrix Admin panel.</p>
<p>Check its login page and other pages, all looks to be extremely good.</p>
<p>In the 2nd part of this video, we will learn how to embed the HTML part of this admin in Laravel 5.6</p>
<p>Part 2: Make Admin Panel in Laravel 5.6</p>
<p>Youtube Tutorial :- <a href="https://www.youtube.com/watch?v=ZchvX0vFfao" rel="nofollow">https://www.youtube.com/watch?v=ZchvX0vFfao</a></p>
<p>Instructions :-</p>
<p>In Part 2, we will copy Matix Admin template files into laravel. We will copy CSS, JS, Images and Fonts from Matrix Admin to laravel file structure making sure that everything will work perfectly when we start actual implementation.</p>
<ol>
<li>Copy CSS Files :- First we will copy CSS file from Matrix admin and for that we will create 2 folders in CSS folder in Laravel :-</li>
</ol>
<p>backend_css and frontend_css</p>
<p>We will create 2 folders for CSS one for backend and one for frontend so that no confusion will arise in future.</p>
<p>So in backend_css folder, we will copy all the CSS files from Matix template.</p>
<ol start="2">
<li>
<p>Copy JS Files :- Now we will copy all JS Files from Matrix admin. And for that also, we will create 2 folders and copy all JS files into backend_js folder.</p>
</li>
<li>
<p>Copy Images :- In the similar way, we will create seperate folders in laravel and copy images.</p>
</li>
<li>
<p>Copy Fonts :- Lastly, we will copy all fonts from Matrix admin in the similar way.</p>
</li>
</ol>
<p>In Part 3, all these CSS, JS, Images and fonts comes into play when we create common files, header/footer files, login and other pages.</p>
<p>Part 3: Make admin panel in Laravel 5.6</p>
<p>Youtube Tutorial :- <a href="https://www.youtube.com/watch?v=ZyOu8ADpjKA" rel="nofollow">https://www.youtube.com/watch?v=ZyOu8ADpjKA</a></p>
<p>Instructions :-</p>
<p>In Part 3, we will create HTML part of login page for Admin by including CSS, JS and Images.</p>
<p>We will create Controller, View and Route for login.</p>
<ol>
<li>Create AdminController file for login and that can also be used later on for other admin pages.</li>
</ol>
<p>We can create AdminController file from comand prompt itself by running command:</p>
<p>php artisan make:controller AdminController</p>
<p>Now our AdminController file is created.</p>
<ol start="2">
<li>In next step, we will open this file and create login function.</li>
</ol>
<p>In login function, we just returning it to view page that we will make it in admin folder in views folder.</p>
<ol start="3">
<li>
<p>So create admin folder in views folder and then create admin_login.blade.php file in which we copy all login.html.</p>
</li>
<li>
<p>Now in admin_login.blade.php file, we will copy complete login.html content located at Matrix admin template.</p>
</li>
<li>
<p>Now Create Route for admin login</p>
</li>
</ol>
<p>Route::get('/admin','AdminController@login');</p>
<ol start="6">
<li>Now we will correct paths of CSS, JS and images to correct the view of the login page of admin.</li>
</ol>
<p>See Login Page of Matrix Admin is ready in our Laravel project.</p>
<p>In Part 4, we will add the functionality in this Login page we have created. So that we can login to admin.</p>
<p>Part 4: Make admin panel in Laravel 5.6</p>
<p>Youtube Tutorial :- <a href="https://www.youtube.com/watch?v=tQwBGK8d2u0" rel="nofollow">https://www.youtube.com/watch?v=tQwBGK8d2u0</a></p>
<p>Instructions :-</p>
<p>In Part 4, We will configure default laravel login process by using Laravel Migration.</p>
<p>Laravel Migrations provide us the mechanism for creating and modifying database tables.</p>
<p>So will will create database first, then create users table with migration and then register/login with it. Later on we will use Matrix Admin login in this process.</p>
<ol>
<li>
<p>Create Database :-  We will create database with name 'Sample'</p>
</li>
<li>
<p>Update .env file :- Now we will update .env file to configure our database connection.</p>
</li>
<li>
<p>Create Users table :- Now we will run some commands to create users table in database and then will create login/register pages also with another auth command.</p>
</li>
</ol>
<p>i) php artisan migrate</p>
<pre><code> Run this command to create users table. If it gives error then check your database settings that you have given in .env file.
</code></pre>
<p>Now see Migration tables automatically created in which users table also there.</p>
<p>Now we will run another command :-</p>
<p>php artisan make:auth</p>
<p>This command will create default login/register pages that we will later use for our admin Matrix login.</p>
<p>Also we need to clear cache once if any error comes..</p>
<p>You just need to run below command in case of error :-</p>
<p>php artisan config:clear</p>
<p>and then restart Laravel and run "php artisan serve" command again.</p>
<p>See in video, we able to login/register successfully now.</p>
<p>In Part 5, we will embed this login system in Matrix Login.</p>
<p>We will use this users table for it and also add one more column for admin/users.</p>
<p>Part 5: Make admin panel in Laravel 5.6</p>
<p>Youtube Tutorial :- <a href="https://www.youtube.com/watch?v=5Oi6NG4bqS8" rel="nofollow">https://www.youtube.com/watch?v=5Oi6NG4bqS8</a></p>
<p>Instructions :-</p>
<p>In this video, we will use default login process that we have created in last video with migration.</p>
<ol>
<li>Update Users table : First of all, update users table that we have created last time. Add one more admin column in it that we will update only for one row.</li>
</ol>
<p>We have updated our earlier record as well with data 1 that we have added in 'admin' column just created in video.</p>
<ol start="2">
<li>Update admin_login.blade.php : Now we will update login form that we have copied from Matrix admin.</li>
</ol>
<p>We will set its action as action = "{{ url('admin') }}"</p>
<p>Also don't forget to add {{ csrf_field }} that will generate security token for validating form. This we will learn in later chapters.</p>
<p>Now update email, password and submit button in form.</p>
<ol start="3">
<li>Now update Route: web.php :-
We will update the admin route that we have created last time with POST method as well.</li>
</ol>
<p>Route::match(['get','post'],'/admin','AdminController@login');</p>
<ol start="4">
<li>Update login function :- Now in AdminController.php we will update our login function.</li>
</ol>
<p>i) First add if condition with $request-&gt;isMethod('post') to check if login form submitted.</p>
<p>ii)And then use Auth::attempt to check if admin email, password is correct. And user actually is admin.</p>
<p>See in video, our login process is working fine now. It will show "Success" message if we login with our earlier user that we have created with email <a href="mailto:amitphpprogrammer@gmail.com">amitphpprogrammer@gmail.com</a> and updated its admin column to 1. But for other users, we have not updated admin column, that's why can't able to login with other users.</p>
<p>In next video, we will login and reach its dashboard.</p>
<p>Part 6: Make admin panel in Laravel 5.6</p>
<p>Youtube Tutorial :- <a href="https://www.youtube.com/watch?v=bzYBu0iez3s" rel="nofollow">https://www.youtube.com/watch?v=bzYBu0iez3s</a></p>
<p>Instructions :-</p>
<p>In Part 6, we will create dashboard page of admin, set its header and footer that we will also use for other pages of admin later on and finally reaches dashboard page from login page.</p>
<ol>
<li>Layout Structure :- Structure is very important now that we will use for all pages of admin and for front as well later on, and that must be clear.</li>
</ol>
<p>First in layouts folder in views folder,</p>
<p>create below folders :-</p>
<p>i) adminLayout
ii) frontLayout</p>
<p>frontLayout folder contains all the common files like header, footer for front that we will see later on.</p>
<p>adminLayout will contain common files for admin; so we will create header, footer for admin and store in this folder.</p>
<p>So, in adminLayout folder, we will create below files :-</p>
<p>admin_design.blade.php (this will contain complete html design of admin, admin_design file will contain header, footer and sidebar)</p>
<p>admin_header.blade.php (admin header part)</p>
<p>admin_footer.blade.php (admin footer part)</p>
<p>admin_sidebar.blade.php (admin sidebar part)</p>
<ol start="2">
<li>Work on admin_design.blade.php:</li>
</ol>
<p>Now we will copy dashboard file from Matrix admin to this file and then include header, footer and sidebar pages.</p>
<p>Now cut header part and paste into admin_header.blade.php that we have created</p>
<p>and include that header file like:</p>
<p>@include('layouts.adminLayout.admin_header')</p>
<p>In the same way, add content in other files; footer and sidebar and include then in admin_design file..</p>
<p>So it must be clear now that admin_design file contains all common files that we have created.</p>
<p>For main content, we will create another file dashboard.blade.php and keep it in admin folder where we earlier kept admin_login.blade.php</p>
<p>And for including dashboard.blade.php file, we will use below code:</p>
<p>@yield('content')</p>
<p>And in dashboard.blade.php, we will use like below:</p>
<p>@extends('layouts.adminLayout.admin_design')
@section('content')
--- content will come---
@endsection</p>
<ol start="3">
<li>Set CSS, JS and copy into related folders :-</li>
</ol>
<p>Now we will set all CSS, JS and images and make sure all files are included in our laravel folders.</p>
<ol start="4">
<li>Create Route for dashboard</li>
</ol>
<p>Route::get('/admin/dashboard','AdminController@dashboard');</p>
<ol start="5">
<li>Create dashboard funtion/method in AdminController.php</li>
</ol>
<p>Now see in video, most of the components are displaying now.. You must debug the missing files from console and need to work to add them all...</p>
